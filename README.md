# SalaryForecaster

Приложение для прогнозирования заработной платы на основе календаря рабочих дней и настроек выплат.

## Описание

SalaryForecaster — это десктопное приложение для Windows, которое помогает планировать и прогнозировать заработную плату. Приложение учитывает рабочие и праздничные дни, настраиваемые даты выплат, а также позволяет добавлять дополнительные выплаты и корректировки.

## Основные возможности

- **Прогнозирование зарплаты**: Расчет будущих выплат на основе календаря рабочих дней
- **Настройка параметров зарплаты**:
  - Размер заработной платы
  - Даты выплаты первой и второй части зарплаты
  - Размер наличной и безналичной части для каждой выплаты
- **Учет праздничных дней**: Автоматическое определение рабочих дней с учетом праздников
- **Дополнительные выплаты**: Возможность добавления и управления дополнительными выплатами по месяцам
- **Корректировки**: Функция корректировки расчетных значений
- **Локализация**: Поддержка русского и английского языков

## Технологии

- **.NET Framework 4.7.2** (Desktop приложение)
- **.NET Standard 1.3** (Core библиотека)
- **WPF** для пользовательского интерфейса
- **MugenMvvmToolkit** для MVVM архитектуры
- **SQLite** для хранения данных
- **Autofac** для dependency injection
- **Newtonsoft.Json** для работы с JSON

## Требования

- Windows 7 или выше
- .NET Framework 4.7.2 или выше
- Visual Studio 2017 или выше (для разработки)

## Использование

1. Запустите приложение
2. Настройте параметры зарплаты в разделе настроек:
   - Укажите размер заработной платы
   - Установите даты выплат первой и второй части
   - Настройте размер наличной и безналичной части
3. Приложение автоматически рассчитает прогноз зарплаты на основе календаря рабочих дней
4. При необходимости добавьте дополнительные выплаты или внесите корректировки

## Разработка

Проект использует архитектуру MVVM с разделением на Core (бизнес-логика) и Desktop (UI). Основные компоненты:

- **CalendarProvider**: Управление календарем и рабочими днями
- **SalaryProvider**: Расчет зарплаты
- **SettingsManager**: Управление настройками приложения
- **DbService**: Работа с базой данных SQLite
